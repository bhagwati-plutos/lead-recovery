# 10. Use Mobile-First Responsive Design

Date: 2026-03-22

## Status

Accepted

## Context

Designing for desktop first leads to bloated CSS and poor mobile experience.

## Decision

We will adopt a mobile-first approach using min-width media queries.

## Consequences

* **Positive:** Better performance on mobile devices.
* **Positive:** Progressive enhancement.
* **Positive:** Cleaner CSS structure.
* **Negative:** Requires mindset shift for some developers.
